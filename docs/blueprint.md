# **App Name**: AgriSense

## Core Features:

- Secure Login: Enable secure user authentication with email user@gmail.com and password user123.
- Automatic Redirection: Automatically redirect users to the User Modules after successful login.
- Crop Scanning Interface: Provide an interface for farmers to upload and manage crop scans.
- AI Diagnosis Tool: Diagnose crop diseases and issues using AI analysis based on uploaded scans and historical data, and including reasoning steps.
- Treatment Recommendations: Offer treatment recommendations generated by AI for diagnosed crop issues.
- Yield Forecasting: Forecast crop yield based on current data and historical trends using AI algorithms.
- Alert System: Notify users of critical issues or upcoming tasks via a personalized alert system.
- User Data Management: Securely manage user data and access control using Firebase Authentication and Firestore.

## Style Guidelines:

- Primary color: Deep green (#386641) to reflect growth, nature, and health. 
- Background color: Very light green (#F0F5F1), a desaturated tone of the primary color.
- Accent color: Earthy brown (#A68A64) to highlight the natural connection to soil and agriculture.
- Headline font: 'Belleza', a humanist sans-serif suitable for headlines.
- Body font: 'Alegreya', a humanist serif to use with 'Belleza'.
- Use nature-inspired icons (e.g., leaves, crops, water droplets) to represent different modules and functions.
- Implement a clean, modular design for easy navigation between crop scanning, diagnosis, and yield forecasting.
- Subtle animations for data loading and transitions to provide a smooth user experience.
